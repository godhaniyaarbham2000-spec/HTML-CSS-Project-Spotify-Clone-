# 🎧 Spotify Clone - Web Player UI

A responsive **Spotify Web Player UI Clone** built using pure **HTML and CSS**.  
This project replicates the modern Spotify desktop interface including sidebar navigation, music cards, and a custom music player UI.

--- 

## 🚀 Features

* 🎨 Modern Spotify-like UI design  
* 📂 Sidebar navigation (Home, Search, Library)  
* 🎶 Music sections (Recently Played, Trending, Artists, Charts)  
* 🎧 Custom music player UI (play, pause, progress bar, volume)  
* 📱 Fully responsive design (mobile + tablet friendly)  
* 🎭 Smooth hover effects and transitions  
* 🔗 Font Awesome icons integration  
* 🌈 Dark theme UI similar to Spotify  

---

## 🛠️ Tech Stack

* HTML5  
* CSS3  
* Font Awesome Icons  
* Google Fonts (Montserrat)

## ⚙️ How to Run

Clone the repository:

```
git clone https://github.com/godhaniyaarbham2000-spec/HTML-CSS-Project-Spotify-Clone.git
```

Open project folder:

```
cd HTML-CSS-Project-Spotify-Clone
```

Run project:

```
Open index.html in your browser
```
